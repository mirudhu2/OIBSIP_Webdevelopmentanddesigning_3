🌡️ Temperature Converter Web App

A simple and interactive Temperature Converter built using HTML, CSS, and JavaScript. This application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin with a modern UI and dark mode support.

✨ Features
🔢 Real-time Temperature Conversion
🌡️ Convert between:
Celsius → Fahrenheit & Kelvin
Fahrenheit → Celsius & Kelvin
Kelvin → Celsius & Fahrenheit
⚠️ Input Validation (handles invalid entries)
🌙 Dark Mode Toggle
🎨 Modern Glassmorphism UI
📱 Responsive Design (works on mobile & desktop)
✨ Smooth Animations

🛠️ Technologies Used
HTML5 – Structure
CSS3 – Styling, animations, and dark mode
JavaScript – Logic and interactivity
